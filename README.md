Book Manager 📚
Aplicação Java desenvolvida como atividade acadêmica para gerenciamento de livros e autores.

🛠 Tecnologias Utilizadas
Java

Spring Boot

Maven

JPA (Hibernate)

PostgreSQL

IntelliJ IDEA

💡 Funcionalidades
Cadastro de livros

Cadastro de autores

Associação entre livros e autores

Visualização de todos os registros
